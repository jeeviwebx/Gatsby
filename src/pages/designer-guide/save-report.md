# Save a Report

This section explains how to save a report after designing it in the Standalone Report Designer.

Click on the `Save` icon in the toolbar.

![Save report menu](/static/assets/standalone-report-designer/images/designer-guide/save-report/save-report-menu.png '#width=150px')

Browse the folder path and click **save**.

![Save report menu](/static/assets/standalone-report-designer/images/designer-guide/save-report/browse-folders.png '#width=420px')
