# Property Navigator

The **Property Navigator** displays the report structure in a hierarchical format, enabling users to quickly locate and select report elements to configure their properties.

![Property Navigator interface showing hierarchical report structure](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/property-navigator-user-interface.png '#width=350px')

## Overview

To open the **Property Navigator**, first access the **Properties Panel** by clicking the `Properties` icon.
![Properties panel icon for opening Property Navigator](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/properties-icon.png '#width=350px')

Next, click the highlighted `Property Navigator` icon. The popup list will appear as shown below.
![Popup list of Property Navigator with report elements](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/property-navigator-popup-list.png '#width=350px')

In the Property Navigator popup list, the report structure is shown in a hierarchical format. Each report element is represented by its name and type. In the example below:

* The icon and the tag (Image) indicate the type of the report item.

* `Logo_Image` is the name of the report item.

![Property Navigator hierarchy view showing element type and name](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/report-element-denotion.png '#width=350px')

You can use the search option to find a specific report element in the Property Navigator popup list. When you search for a report element by name, the results are displayed in a **parent-child hierarchical** format.
![Search results in Property Navigator with parent-child hierarchy](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/search-with-match.png '#width=350px')

If no results are found, a `No matches found` message will appear.
![Property Navigator search error message](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/search-with-no-match.png '#width=350px')

Selecting a report section (such as Report, Header, Body, or Footer) or a specific report item in the Property Navigator displays its properties in the Properties Panel.
![Selecting report items in Property Navigator highlights properties](/static/assets/on-premise/images/report-designer/compose-report/property-navigator/report-item-selection-tick.png '#width=350px')

To close the Property Navigator popup list, click anywhere outside the popup area.
