# Preview a Report

This section explains how to preview a report after successfully designing it in the Standalone Report Designer.

1. Click on the **Preview** button in the toolbar.
   ![Preview button in report designer](/static/assets/standalone-report-designer/images/designer-guide/preview-report/option-in-toolbar.png '#width=200px')

2. Now, the report will be previewed using the built-in Report Viewer.
   ![Report preview using Report Viewer](/static/assets/standalone-report-designer/images/designer-guide/preview-report/preview-in-report-viewer.png '#width=600px')
