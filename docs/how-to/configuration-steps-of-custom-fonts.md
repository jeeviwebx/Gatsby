# How to configure the custom fonts in Standalone Report Designer

Configure the custom fonts as shown in the following steps.

1. Place the required font file in the following location.

   `{Installed Location}\Bold Reports\Report Designer\CustomFonts`

   ![Place Custom Fonts](/static/assets/standalone-report-designer/images/how-to/custom-fonts/place-custom-fonts.png '#width=600px')

2. Configure your custom fonts into the config.json file.

   ![Configure Downloaded File](/static/assets/standalone-report-designer/images/how-to/custom-fonts/configure-downloaded-file.png '#width=335px')

3. Close the application and reopen it to load your custom fonts in the designer. The newly added font is added in the designer, as shown in the following image.

   ![Custom Font Loaded](/static/assets/standalone-report-designer/images/how-to/custom-fonts/custom-font-loaded.png '#width=385px')

4. Design the report with newly added custom fonts.

   ![Design View](/static/assets/standalone-report-designer/images/how-to/custom-fonts/design-view.png '#width=550px')

5. Then, preview the report to see the newly added custom font style.

   ![Preview](/static/assets/standalone-report-designer/images/how-to/custom-fonts/preview-report.png '#width=415px')
