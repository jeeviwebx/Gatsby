# Open report

This section explains how to open an existing reports in the Standalone Report Designer.

Click on the `Open` icon in the toolbar.

![Open report menu](/static/assets/standalone-report-designer/images/designer-guide/open-report/open-report-menu.png '#width=150px')

Browse the report path and open the report.

![Browse report](/static/assets/standalone-report-designer/images/designer-guide/open-report/browse-report.png '#width=450px')

Now, the respective report will be opened in the designer.

![Design surface with report](/static/assets/standalone-report-designer/images/designer-guide/open-report/final-output.png)
